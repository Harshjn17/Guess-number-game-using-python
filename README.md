# Guess-number-game-using-python

This is a simple Number Guessing Game built using Python.
The program generates a fixed secret number, and the user keeps guessing until the correct number is entered.
After each guess, the program gives feedback whether the guess is too high, too low, or correct.
This project helps beginners practice loops, conditions, input/output, and variables.
